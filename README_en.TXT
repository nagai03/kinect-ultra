kinect-ultra V0.1
Author: Tomoto S. Washio
Release Date: 1/9/2011
Project URL: http://code.google.com/p/kinect-ultra/

1. Prerequisitive

(1) You already have Kinect, right?
(2) This product is supposed to run on sevral Windows platforms although
    it is only tested on Windows XP Pro SP3 at this moment.
(3) Your computer should have sufficiently fast CPU and graphic cards.
    The minimum requirement is unknown, but it runs well on Core i7 920
    with GeForce GTX 285.

2. Installation

(1) If you have not yet done, download OpenNI and install it.
    http://www.openni.org/downloadfiles/openni-binaries/stable/26-openni-alpha-build-for-windows-v1-0-0/download

(2) If you have not yet done, download OpenNI-compatible Kinect driver
    and install it.
    https://github.com/avin2/SensorKinect 

    If you already have OpenKinect or other drivers installed, you will
    need to switch those drivers.

(3) Unpack this archive anywhere you like.

3. How to Use

(1) Just run kinect-ultra.exe.
(2) See the demo at:
    http://www.youtube.com/watch?v=RUG-Uvq-J-w (English)
    http://www.youtube.com/watch?v=eCbURRDUUdI (Japanese).

5. Build

If you want to build this program yourself or modify, go to the project
page, download the source code, and open build/kinect-ultra.sln with
Visual C++ 2008.
